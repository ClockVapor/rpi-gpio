#rpi_gpio v0.1.0

Ruby conversion of [RPi.GPIO Python module](https://pypi.python.org/pypi/RPi.GPIO)

##Features

- Binary GPIO pin input/output
- Software PWM output for GPIO pins (written in C for speed)
